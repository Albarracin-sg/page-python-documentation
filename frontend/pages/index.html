Sintaxis básica de Python

Python es conocido por su sintaxis limpia y legible. A diferencia de otros lenguajes de programación, utiliza la indentación para definir bloques de código.


                        # Este es un comentario en Python
                        print("Hola, mundo!") # Imprime un mensaje

                        # La indentación es importante
                        if 5 > 2:
                            print("Cinco es mayor que dos")  # Esta línea está indentada

Python es sensible a mayúsculas y minúsculas, lo que significa que variable y Variable son consideradas diferentes.

Variables y tipos de datos

En Python, no es necesario declarar el tipo de variable. El tipo se asigna automáticamente cuando se asigna un valor.

# Variables y tipos básicos
                        nombre = "Ana"  # String (cadena de texto)
                        edad = 25       # Integer (entero)
                        altura = 1.75   # Float (decimal)
                        es_estudiante = True  # Boolean (booleano)

                        # Verificar el tipo de una variable
                        print(type(nombre))  # 
                        print(type(edad))    #

Tipos de datos principales:

  • Strings (str): Para texto y caracteres
  • Integers (int): Números enteros
  • Floats (float): Números decimales
  • Booleans (bool): Valores verdadero (True) o falso (False)
  • Lists (list): Colecciones ordenadas y modificables
  • Tuples (tuple): Colecciones ordenadas e inmutables
  • Dictionaries (dict): Colecciones desordenadas con pares clave-valor
  • Sets (set): Colecciones desordenadas sin elementos duplicados

Estructuras de control

Las estructuras de control permiten modificar el flujo de ejecución de un programa.

Condicionales (if, elif, else)


                        # Estructura condicional básica
                        edad = 18

                        if edad < 18:
                            print("Eres menor de edad")
                        elif edad == 18:
                            print("Acabas de cumplir la mayoría de edad")
                        else:
                            print("Eres mayor de edad")

Bucles


                        # Bucle for
                        for i in range(5):
                            print(i)  # Imprime 0, 1, 2, 3, 4

                        # Bucle while
                        contador = 0
                        while contador < 5:
                            print(contador)
                            contador += 1  # Incrementa el contador

Funciones

Las funciones son bloques de código reutilizables que se ejecutan cuando son llamadas.


                        # Definición de una función simple
                        def saludar(nombre):
                            return f"Hola, {nombre}!"

                        # Llamada a la función
                        mensaje = saludar("María")
                        print(mensaje)  # Imprime: Hola, María!

                        # Función con parámetros por defecto
                        def presentar(nombre, edad=25):
                            return f"{nombre} tiene {edad} años."

                        print(presentar("Juan"))  # Imprime: Juan tiene 25 años.
                        print(presentar("Ana", 30))  # Imprime: Ana tiene 30 años.

Funciones lambda

Las funciones lambda son pequeñas funciones anónimas definidas con la palabra clave lambda.


                        # Función lambda para duplicar un número
                        duplicar = lambda x: x * 2

                        print(duplicar(5))  # Imprime: 10

Arrays en Python

En Python, los arrays se pueden manejar de varias formas, siendo las listas la estructura más básica y NumPy para cálculos científicos.

Listas como Arrays Básicos


                        # Crear una lista (array básico)
                        numeros = [1, 2, 3, 4, 5]
                        
                        # Operaciones básicas
                        print(numeros[0])    # Acceder a un elemento
                        print(len(numeros))  # Longitud del array
                        numeros.append(6)    # Agregar un elemento
                        numeros.remove(3)    # Eliminar un elemento
                        
                        # Arrays multidimensionales
                        matriz = [
                            [1, 2, 3],
                            [4, 5, 6],
                            [7, 8, 9]
                        ]
                        print(matriz[1][2])  # Acceder a un elemento específico: 6

NumPy: Librería para Computación Numérica

NumPy es fundamental para cálculos científicos y computación numérica en Python. Es esencial para trabajar con arrays de manera eficiente.


                        # Importar NumPy
                        import numpy as np
                        
                        # Crear arrays NumPy
                        array_1d = np.array([1, 2, 3, 4, 5])
                        array_2d = np.array([[1, 2, 3], [4, 5, 6]])
                        
                        # Operaciones matemáticas
                        print(array_1d * 2)  # Multiplicación elemento por elemento
                        print(np.mean(array_1d))  # Media
                        print(np.median(array_1d))  # Mediana
                        
                        # Funciones de generación de arrays
                        zeros = np.zeros((3, 3))  # Matriz de ceros
                        ones = np.ones((2, 2))    # Matriz de unos
                        rango = np.arange(0, 10, 2)  # Array con rango [0, 2, 4, 6, 8]
                        
                        # Reshaping de arrays
                        reshape = np.arange(12).reshape(3, 4)

Ventajas de NumPy

  • Operaciones matemáticas más rápidas
  • Memoria más eficiente
  • Soporte para arrays multidimensionales
  • Herramienta fundamental para ciencia de datos

Cálculos Estadísticos Básicos

Implementación de medidas estadísticas usando NumPy y funciones propias.


                        import numpy as np
                        from statistics import mode
                        
                        # Conjunto de datos
                        datos = [4, 5, 2, 7, 9, 3, 5, 5, 1, 6]
                        
                        # Funciones estadísticas con NumPy
                        media = np.mean(datos)
                        mediana = np.median(datos)
                        
                        # Calcular la moda
                        def calcular_moda(datos):
                            try:
                                moda = mode(datos)
                            except StatisticsError:
                                # Si hay múltiples modas, toma la primera
                                moda = max(set(datos), key=datos.count)
                            return moda
                        
                        # Calcular estadísticas
                        moda = calcular_moda(datos)
                        print(f"Moda: {moda}")
                        print(f"Media: {media}")
                        print(f"Mediana: {mediana}")
                        
                        # Desviación estándar
                        desviacion = np.std(datos)
                        print(f"Desviación Estándar: {desviacion}")

  • from: Para importar módulos específicos
  • def: Para definir funciones personalizadas
  • return: Para devolver valores de funciones
  • try/except: Para manejo de errores

Calculadora Simple

Este ejemplo muestra cómo crear una calculadora básica utilizando funciones:


                # Calculadora básica en Python
                def suma(a, b):
                    return a + b

                def resta(a, b):
                    return a - b

                def multiplicacion(a, b):
                    return a * b

                def division(a, b):
                    if b == 0:
                        return "Error: División por cero"
                    return a / b

                # Menú de opciones
                print("Calculadora Simple")
                print("1. Suma")
                print("2. Resta")
                print("3. Multiplicación")
                print("4. División")

                opcion = int(input("Seleccione una operación (1-4): "))
                num1 = float(input("Ingrese el primer número: "))
                num2 = float(input("Ingrese el segundo número: "))

                if opcion == 1:
                    print(f"Resultado: {suma(num1, num2)}")
                elif opcion == 2:
                    print(f"Resultado: {resta(num1, num2)}")
                elif opcion == 3:
                    print(f"Resultado: {multiplicacion(num1, num2)}")
                elif opcion == 4:
                    print(f"Resultado: {division(num1, num2)}")
                else:
                    print("Opción no válida")

Lista de Tareas

Un ejemplo de cómo manejar una lista de tareas usando listas y bucles:


                # Aplicación simple de lista de tareas
                tareas = []

                def agregar_tarea(tarea):
                    tareas.append(tarea)
                    print(f"Tarea '{tarea}' agregada.")

                def ver_tareas():
                    if not tareas:
                        print("No hay tareas pendientes.")
                    else:
                        print("Tareas pendientes:")
                        for i, tarea in enumerate(tareas, 1):
                            print(f"{i}. {tarea}")

                def eliminar_tarea(indice):
                    if 1 <= indice <= len(tareas):
                        tarea_eliminada = tareas.pop(indice - 1)
                        print(f"Tarea '{tarea_eliminada}' eliminada.")
                    else:
                        print("Índice no válido.")

                # Ejemplo de uso
                agregar_tarea("Estudiar Python")
                agregar_tarea("Hacer ejercicio")
                ver_tareas()
                eliminar_tarea(1)
                ver_tareas()

Análisis de Texto

Este ejemplo muestra cómo analizar un texto para contar palabras y caracteres:


                # Analizador de texto simple
                def analizar_texto(texto):
                    # Eliminar espacios adicionales
                    texto = texto.strip()
                    
                    # Contar caracteres (incluidos espacios)
                    num_caracteres = len(texto)
                    
                    # Contar palabras
                    palabras = texto.split()
                    num_palabras = len(palabras)
                    
                    # Contar frecuencia de palabras
                    frecuencia = {}
                    for palabra in palabras:
                        palabra = palabra.lower()
                        if palabra in frecuencia:
                            frecuencia[palabra] += 1
                        else:
                            frecuencia[palabra] = 1
                    
                    # Palabra más común
                    palabra_comun = max(frecuencia.items(), key=lambda x: x[1])
                    
                    return {
                        "caracteres": num_caracteres,
                        "palabras": num_palabras,
                        "frecuencia": frecuencia,
                        "palabra_comun": palabra_comun
                    }

                # Ejemplo de uso
                texto_ejemplo = """
                Python es un lenguaje de programación interpretado cuya filosofía hace hincapié 
                en la legibilidad de su código. Es un lenguaje interpretado, dinámico y multiplataforma.
                Python es un lenguaje muy popular para la ciencia de datos y la inteligencia artificial.
                """

                resultado = analizar_texto(texto_ejemplo)
                print(f"Caracteres: {resultado['caracteres']}")
                print(f"Palabras: {resultado['palabras']}")
                print(f"Palabra más común: '{resultado['palabra_comun'][0]}' ({resultado['palabra_comun'][1]} veces)")
